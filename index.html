    Oque é Python?

    Python é uma linguagem de programação de alto nível,
    interpretada e de propósito geral.
    Foi criada por Guido van Rossum e lançada em 1991.
    Python é conhecida por sua sintaxe clara e legibilidade,
    o que a torna uma escolha popular para iniciantes em programação.

    Python é usado em uma variedade de aplicações,
    desde desenvolvimento web, ciência de dados,
    automação de tarefas, inteligência artificial,
    até jogos e muito mais. Sua versatilidade,
    facilidade de uso e ampla adoção fazem do Python uma das linguagens de programação mais populares atualmente.
